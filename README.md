# NodeUIM

CA UIM NodeJS interface to work with pu.exe in a full async way.

## Requirement 

- NodeJS 7.7.0 or higher

## Installation 

```
npm install nodeuim [--save]
```

## Documentation 

Find API & Documentations on the wiki page - [Here](https://github.com/fraxken/NodeUIM/wiki)
